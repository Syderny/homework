# Chapter 09

## 1. 游戏涉及哪些类

* 指引词
* 操作按钮：    GO
* 控制对象：    牧师与魔鬼
* 计时牌

## 2. 游戏中有哪些对象，各几个
* 船：1
* 牧师：3
* 魔鬼：3
* 倒计时牌：1
* go控制按键：1

## 3. 类和对象的区别是什么

类是一类拥有相似属性的对象的集合。比如可控制上下船对象(类)拥有牧师和魔鬼(对象)。

## 4. 游戏中的魔鬼，有哪些属性和方法？

属性: 颜色, 长宽高, 位置, 动作;
方法: 上\下船, 杀死牧师;

## 5. 用文字（伪代码）描述魔鬼与其他对象沟通的过程
```basic
IF onclick时魔鬼在岸上 THEN
    IF 船上人数 < 2 THEN
        魔鬼上船
    ELSE
        魔鬼不动
    END IF
END IF

IF onclick时魔鬼在船上 THEN
    魔鬼上岸
END IF
```
